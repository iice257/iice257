## What I've Built

### [PowerGrid](https://github.com/iice257/PowergridApp)
A mobile web app for tracking real-time electricity availability in Nigeria. It combines crowdsourced reporting, predictive thinking, and data-efficient design for users dealing with intermittent connectivity and low-end hardware.

This project sharpened my thinking around performance constraints, offline-aware product decisions, and building for users that many software teams overlook.

### [FormMate](https://github.com/iice257/FormMate)

An AI-assisted form companion designed to reduce the friction of repetitive form filling. It combines conversational assistance, autofill-style workflows, and a clean product-focused interface to help users complete forms faster and with less mental load.

### [Pulse](https://github.com/iice257/Pulse)

A mobile-first product focused on connectivity, responsiveness, and efficient user experience under real-world constraints. It reflects my interest in building systems that stay lightweight, reliable, and practical even when network quality and device capability are limited.
